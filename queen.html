Problem Description

Given an N x N grid representing a chessboard, your task is to find a solution that satisfies certain conditions. Each cell of the chessboard can be either empty or contain a piece.

Constraints:

  • The grid is of size N x N, where N is a positive integer.
  • Each cell of the grid can either be empty (denoted by '.') or contain a piece (denoted by 'P').

Conditions:

  • No Two Pieces in the Same Row or Column: There can be at most one piece in each row and each column of the chessboard.
  • Diagonal Constraints: No two pieces can attack each other diagonally. This means that there should be no two pieces such that they are on the same diagonal line (either main diagonal or secondary diagonal).

Objective:

Your goal is to find a configuration of pieces on the chessboard that satisfies the above conditions. If a solution exists, return the grid configuration; otherwise, return an empty grid.
